浅色冷钢蓝的 3 个方向

全部保持当前排版不变，只看浅色主题本身。区别主要在“偏设备、偏洁净、还是偏中性灰蓝”。

A
雾银蓝 / 设备台
最像工业检测设备面板。浅色，但不会发白发空，金属感最强。
Recommended
字体：Segoe UI + Bahnschrift。冷静、稳、偏控制台。
B
冰白蓝 / 洁净实验室
更亮、更干净、更接近实验室终端。看起来最轻，但也最容易偏“太医疗”。
字体：Segoe UI。偏轻、偏洁净、偏现代。
C
灰钢蓝 / 中性终端
浅色里最克制，蓝味更淡，偏灰钢。没那么冷，也不偏暖，最容易长期看。
字体：Bahnschrift + Segoe UI。偏稳、偏中性。
